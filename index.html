css text property

CSS text property has six different types

The six property are:
1.text color
2.text alignment
3.text decoration
4.text transformation
5.text spacing
6.text shadow

Definition

1.Text color has three ways like
a.color name
b. HEX value
c. RGB
2.Text alignment - a text has default alignment like right center left
3.Text decoration has four ways like a.overline
b.linethrough
c.underline
d. none
